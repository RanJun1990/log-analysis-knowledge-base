# ログイン失敗・停止 解析テンプレート（日本語）

以下の条件で、ログイン障害を重点分析してください。

## 入力
- ログパス: {{LOG_PATH}}
- 対象時間帯: {{START_TIME}} ～ {{END_TIME}}
- 同時利用人数: {{CONCURRENT_USERS}}
- 事象: {{INCIDENT_DESC}} 例: 「同時ログイン時に止まる」

## 解析ポイント（必須）
- LoginFilter の start / finish 件数差
- ERROR / WARN / Exception / timeout の件数と時刻
- 遅延（avg / p95 / max）
- 分単位ログイン件数推移
- 分単位スレッド数推移
- 遅延上位処理と関連モジュール

## 判定ルール
- 明示的エラー多発 → 機能失敗型
- start=finish かつ高遅延 → 輻輳・待ち行列型
- 同時にDB遅延ログ増加 → DB/SQLボトルネック疑い

## 出力（必須）
- 事象要約（3行）
- 証拠一覧（数値）
- 根因候補Top3
- 再発防止アクション（短期/中期）
